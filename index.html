Rumus Luas Segitiga:

L = 1/2 × a × t

Dengan keterangan:

  • L = Luas
  • a = Panjang alas
  • t = Tinggi

Rumus Keliling Segitiga:

K = A + B + C

Dengan keterangan:

  • K = Keliling
  • A = Sisi A
  • B = Sisi B
  • C = Sisi C
